In diesem Abschnitt wird der Kernel eines Betriebssystems erklärt.

Kernel Basics

Ein Kernel ist ein Stück low-level Software, das direkten Speicherzugriff hat.

Sie ist für verschiedene Funktionen verantwortlich, wie zum Beispiel:

  • Schnittstelle für Software Anwendungen
  • Kontrolle des CPU Zugriffs
  • Warteschlangen verwalten bei schwachen Systemen
  • Input/Output verwalten

Architekturen

Kernels gibt es in verschiedenen Architekturen:

  • Der Microkernel
  • Der Monolithische Kernel
  • Der Hybridkernel
