Linux

Dieser Kernel ist vollständig Open Source, das heißt, jeder kann den Quelltext einsehen, ihn bearbeiten und selber verwenden. Es wurde erstmals im Jahr 1991 vom finnischen Informatiker Linus Torvalds als persönliches Projekt entwickelt.

Es gibt nicht „ein“ Linux. Wenn jemand über Linux redet, meint er höchstwahrscheinlich die Familie von Betriebssystemen, die auf den Linux Kernel bauen, sowie GNU Software.

Bekannte Linux Distributionen:

  • Ubuntu
  • Debian
  • Arch Linux
  • openSUSE Leap/Tumbleweed
  • Fedora

Microsoft Windows

Wahrscheinlich das bekannteste Betriebssystem überhaupt. Windows erfreut sich großer Soft- und Hardwarekompatibilität, aber leider auch an viel proprietärer Software und alten Systemarchitekturen, wie dem erstmals 1993 entwickeltem NT Kernel.

Hier gibt es deutlich weniger Auswahlmöglichkeiten als mit Linux. Meistens hat man nur die Option das neueste Windows zu nutzen, oder, wenn man will, auch eine alte Version, aber diese kriegen oftmals leider keine Updates mehr, was den Gebrauch dieser Betriebssysteme im Internet riskant machen kann.

Die wahrscheinlich bekanntesten Windows Versionen:

  • Windows XP
  • Windows 7
  • Windows 10

Apple's MacOS

Dieses Betriebssystem ist ausschließlich auf Apple Computern zu nutzen, zumindest offiziell. Es ist sehr beliebt bei Leuten, die viel kreativ arbeiten. Selbst im Vergleich zu Windows ist MacOS ziemlich restriktiv, da man es nicht beliebig auf jedem PC installieren kann.

Ältere Versionen dieses Betriebssystems sind nur nutzbar, wenn man ein älteres Gerät hat. Allerdings gilt hier nach wie vor: keine Sicherheitsupdates bedeutet mehr Risiken.

Zusammenfassung:

  • Linux: Ein quelloffener, monolithischer Kernel
  • Windows NT: Ein proprietärer Hybridkernel mit guter Soft- und Hardwarekompatibilität
  • MacOS XNU: Ein proprietärer Hybridkernel der speziell für Apple's Computer gemacht wurde
